Create a complete React + Vite multi-page website for a sewing salon called "Alex's Design".

## Tech Stack
- React + Vite
- React Router v6
- Plain CSS (no Tailwind, no CSS Modules — one global.css + per-component CSS files)
- Google Fonts

## Typography
Import these fonts from Google Fonts in index.html:
  - "Climate Crisis" → display font for all headings and the logo (variable font, use font-variation-settings)
  - "DM Sans" → body text, nav links, paragraphs
```html
<link href="https://fonts.googleapis.com/css2?family=Climate+Crisis&family=DM+Sans:wght@300;400;500&display=swap" rel="stylesheet">
```

## Color Palette (extracted from dark UI screenshot)
Use these exact CSS variables in :root {}:
```css
:root {
  --bg-deep:        #0D1117;   /* near-black background */
  --bg-card:        #161B27;   /* card/panel surfaces */
  --bg-card-hover:  #1E2535;   /* card hover state */
  --border:         #2A3147;   /* subtle borders */
  --accent-purple:  #7C5CFC;   /* primary purple accent */
  --accent-teal:    #22C7A9;   /* teal/green accent (icons, CTAs) */
  --accent-pink:    #C45CF5;   /* violet/pink glow */
  --text-primary:   #EAEEF6;   /* main text */
  --text-muted:     #7A849A;   /* muted/secondary text */
}
```

## Global CSS Rules
- background: var(--bg-deep) on body
- All h1, h2, h3: font-family: 'Climate Crisis', cursive; color: var(--text-primary)
- Body text: font-family: 'DM Sans', sans-serif
- Cards: background var(--bg-card), border 1px solid var(--border), border-radius 14px
- Buttons: background linear-gradient(135deg, var(--accent-purple), var(--accent-pink)), color white, border-radius 8px, padding 12px 28px, font DM Sans 500
- Smooth scroll: html { scroll-behavior: smooth }

## Animations
- Hero: h1 fades in + slides up on mount (CSS @keyframes, 0.8s)
- Cards: fade-in on scroll via IntersectionObserver (add 'visible' class → opacity 1, translateY 0)
- Gradient blobs in hero: slow infinite floating animation
- Nav links: color transition 0.2s on hover
